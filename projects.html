Quiz App

An application that utilizes the ability of jQuery. Simple quiz app that ask a series of questions, responds each question with a reaction if the user got the answer correct or wrong while calculating the score and letting the user know which question they're currently on. The application introduced me to the use of jQuery for DOM manipulation and traversal. The end result displays the total score and gives them a message depending on the score.

Tech used: HTML5, CSS3, JavaScript, jQuery
